ğŸ“– Getting Started¶

Prerequisites¶

  • Python 3.11+
  • Node.js 18+
  • Git

Installation¶

  1. Clone the repository: git clone https://github.com/asap-programmer/static-site-python.git cd static-site-python

  2. Install Python dependencies: pip install -r requirements.txt

  3. Install Node.js dependencies:

npm install

Development¶

Start the development server:

mkdocs serve

Build the site:

mkdocs build

Process CSS with PostCSS:

npm run build:css

ğŸ¨ Customization¶

The theme is fully customizable through:

  • HTML Templates: Located in custom_theme/
  • CSS Styles: Custom styles in custom_theme/assets/css/
  • JavaScript: Interactive features in custom_theme/assets/js/
  • Configuration: Site settings in mkdocs.yml
